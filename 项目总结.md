## 3 软件编译运行方式

首先，将本项目的文件克隆到本地并进入到该文件夹：

```bash
git clone https://github.com/RTCSD2016/air_hockey.git
cd air_hockey
```

然后构建：

```bash
cmake .
make
```

生成文件在`build`目录下：

```bash
cd build
./air_hockey           # 主程序
./unit_test            # 启动单元测试
```
